### 4. 인수 배열(arguments)

-arguments 배열은 함수를 호출할 때 전달된 모든 인수를 접근할 수 있게 한다.

-arguments는 실제 배열은 아니고 배열 같은 객체이다.

```javascript 1.8
var sum = function() {
    var i, sum = 0;
    for (i = 0 ; i < arguments.length ; i++) sum += arguments[i];
    return sum;
}
    
console.log(sum(4,8,12,16,23,42));  // 108
```
