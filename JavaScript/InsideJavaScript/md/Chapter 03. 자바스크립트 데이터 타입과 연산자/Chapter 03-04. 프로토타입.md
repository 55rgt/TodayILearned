## 프로토타입
자바스크립트의 모든 객체는 자신의 부모 역할을 하는 객체와 연결되어 있다.

이러한 부모 객체를 프로토타입 객체(프로토타입)라고 부른다.

```javascript 1.8
var foo = {
  name: 'foo',
  age: 30  
};
    
// foo 객체의 프로토타입에 메서드가 정의되어 있으므로 결과가 출력된다.
foo.toString();
console.dir(foo);
```

모든 객체는 자신의 프로토타입을 가리키는 **[[Prototype]]이라는 숨겨진 프로퍼티**를 가진다.

크롬에서는 **__proto__프로퍼티** 형태로 구현되어 있다.
